# Large Language Models (LLMs): From Fundamentals to Advanced Architectures

A beginner-friendly repository that explains the concepts, architectures, training techniques, and optimization methods used in modern Large Language Models (LLMs). Each chapter combines theory with practical Python implementations and Jupyter notebooks.
